Linked List Interactive Program

Files:

- linked_list.c : interactive program to create, insert, delete, display a singly linked list

Compile (PowerShell / Windows):

- With GCC (MinGW or similar):
  gcc linked_list.c -o linked_list.exe
- With MSVC (cl):
  cl linked_list.c

Run:

- In PowerShell:
  ./linked_list.exe

Quick test input file `test_input.txt` is provided to exercise some operations non-interactively using redirection.
